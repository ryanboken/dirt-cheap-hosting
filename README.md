# Dirt cheap hosting

Example github action and html for [dirt-cheap hosting video](https://www.youtube.com/@Ryan_Boken). A detailed writeup is likely available on [Substack](https://ryanboken.substack.com/). 


## Steps
1. Set up your GitHub repository
2. Create AWS Infra
3. Create IAM Policy
4. Create Identity Provider
5. Create IAM Role
6. Update S3 Permissions
7. Add AWS Config Values to Github Secrets
8. Create a GitHub Actions workflow file 
9. Test the workflow!
